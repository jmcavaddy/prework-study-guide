Prework Study Guide

HTML

  • Head element contains info about the website and instructions for the browser displaying the document
  • Body element contains the content displayed to the user
  • An element that is indented underneath another element is the child element of the parent element it is under

CSS

  • link rel="stylesheet" href=filepath of style.css of interest in the header of the HTML doc links the HTML and CSS stylesheet
  • Margin indicates how much space there is outside of an element and will effect other elements position on the page
  • Padding indicates how much space should go around the content of an element

Git

  • git status reports the working branch
  • git checkout -b branchname creates and moves the user to a new working branch
  • git pull origin main merges the version of the main branch that is in our git repo with the main branch that is on our local machine

JavaScript

  • A variable is a named contatiner that stores many formats of data
  • Control flow is the order in which a computer executes code in a script
  • function followed by a name in lowerCamelCase will initiate a function
